★ 手机淘宝

淘宝母婴客户端

负责母婴客户端的宝宝库, Timeline 以及 HTTP API 层的设计和开发.

* Deal Push 

Deal Push 使用 Node.js 编写, 通过爬虫(cheerio)抓取各大优惠推荐网站(包括一淘网, 网易惠惠, 什么值得买, 北美省钱快报等将近20个网站), 通过关键字(可在 Web 端 / iOS 客户端添加)检索出用户想要的信息并保存到数据库, 通过邮件发送, 或者通过 APNS 发送到 iOS 客户端。测试账号 friend 密码 myfriend。

* Paste Push 

很简单的功能，就是通过 PC Web 端（并包装成Chrome插件方便使用）发送文本到 iPhone, 因为 iOS 7 支持 background refresh，所以利用这个特性可以让 App 在收到 Push 后自动粘贴数据到剪切板。

* KLMessager 

使用 Socket.IO 的一个 Demo，Messager 顾名思义就是聊天，支持 Web 端和 iPhone 端。账号必须为 levey 或者 karin。

* Remain

Remain 是一个为极简主义者设计的一个记事工具。

*Coder x Coder

Coder x Coder（缩写CXC），名称借鉴Hunter x Hunter（全职猎人）。
一个简洁的 iOS 、OS X 平台下的社区网站，使用 Ruby on Rails。

* PhotoString

PhotoString 是一个图片搭上文字，并可以竖向拼接的应用程式，有4种样式可选，可为文字选择字体，有4种分辨率可供导出，并且可以分享到 Facebook、Twitter、Tumblr。

相格

一个拼接图片的 App，用户可以选择几张的照片（或者拍几张），拼合起来，摇一摇还可以打乱顺序重新生成图片，可切换相框和背景。可发送到腾讯微博,新浪微博和人人网。

Shange

英文版的“相格”，可以发送到 Twitter，Facebook，Tumblr。

手机工坊

匹配 iPhone 的机型，然后切换背壳的图片，能从服务器上下载更多图片，让用户填写订单后，提交到服务器。

品客

都市快报

都市快报的 iPhone 版,新闻类的 App.

Handbook

母公司日本 Infoteria 的一个长期项目，看书软件，比较大的一个项目，有后台编辑网站，我添加过一些功能和修复过一些 Bug.

Power GIS

杭州电力局的 GIS 系统，显示，搜索杭州电力局的设备（如配电站，电缆箱，变电站等），并在MapView上显示，并作性能优化，如移除不在屏幕内的的自定义大头针(annotation)，使 App 在 iPad 上有良好的体验。

台州联通汽车手机生活

台州联通的一个项目，页面比较多，但是通过 OO 的设计可以重用很多部分，所以也还比较简单。

哈根达斯点单客户端

哈根达斯的点餐应用，用这个 demo 成功拿下这个项目。iPod touch 4上用的，供服务员扫描菜单上的二维码，然后发送订单给服务器，自动完成订单。

蓝牙猜拳

这又是一个小 App,其实此类应用早已泛滥，这是我刚开始接触 iPhone 开发自己做个玩的。顾名思义就是猜拳，当然，首先用蓝牙配对，然后用户就可以甩动iPhone（重力感应）猜拳了，有统计分数，隐藏作弊等功能。
